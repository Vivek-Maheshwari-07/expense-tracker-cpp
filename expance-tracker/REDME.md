# 💸 Expense Tracker in C++

A simple console-based **Expense Tracker** built using **C++**.  
This project helps you manage and track your daily expenses by storing them with details like date, category, amount, and note.

---

## 📌 Features

- 📅 Add expenses with:
  - Date (DD/MM/YYYY)
  - Category (like Food, Travel, etc.)
  - Amount
  - Description/Note
- 📂 View all saved expenses
- 📊 Show total amount spent
- 🔍 Filter expenses by category
- 💾 Data is stored in a file named `expenses.txt` for persistence

---

## 🚀 How to Run

### 🛠️ Compile

Use any C++ compiler. For example, with g++:
