# Expense Tracker (C++)

A simple console-based Expense Tracker built using C++.
This application helps users record, store, and manage their daily expenses through a menu-driven interface.

The project was created to practice core C++ programming concepts such as file handling, structured programming, and user input management.

---

# Project Overview

Managing daily expenses is important for understanding spending habits and maintaining financial discipline. Many people find it difficult to keep track of where their money is going.

This project provides a simple command-line solution where users can add expenses and review previously recorded expenses.

The program stores expense data so that users can track their spending history easily.

---

# Features

• Add new expense records
• View stored expenses
• Save expense data using file handling
• Simple and interactive menu system
• Console based user interface
• Lightweight and easy to run

---

# How the Program Works

The application runs in the terminal and presents a menu-driven interface.

Typical workflow:

1. User runs the program
2. A menu appears with different options
3. User selects an option
4. The program performs the requested action
5. Expense data is stored or displayed

Example flow:

Add Expense → Save Data → View Expense List
